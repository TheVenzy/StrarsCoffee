# Веб-приложение для обслуживания клиентов кофейни "Звездная кофейня"

## Описание проекта

Веб-приложение для автоматизации бронирования столиков и управления меню кофейни. Разработано с использованием Django и PostgreSQL.

### Основные функции:
- Онлайн-бронирование столиков с проверкой доступности
- Просмотр меню с категориями (кофе, чай, десерты, сезонное)
- Отображение акций и сезонных предложений
- Административная панель для управления бронированиями и контентом

---

## Требования к системе

- **Python** 3.10 или выше
- **PostgreSQL** 14 или выше
- **Git** (для клонирования репозитория)
- **Операционная система**: Windows / Linux / macOS
